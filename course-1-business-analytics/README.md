# Course 1: Data Analytics for Business (DA101)

## Assignment: 2Market Marketing Analysis

Analysed marketing campaign data for 2Market, a fictional retail company, to identify customer spending patterns and campaign effectiveness. The analysis used SQL for data exploration and Tableau for interactive visualisation.

### Key Findings

- Identified spending patterns across countries, marital statuses, and product categories
- Built an interactive Tableau dashboard for stakeholder exploration
- Developed SQL queries for customer segmentation and campaign ROI analysis

### Assignment Files

| File | Description |
|------|-------------|
| `assignment-2market/sql-queries/` | 9 SQL query files covering table creation, joins, aggregations, and campaign analysis |
| `assignment-2market/workbooks/` | Tableau dashboards and supporting data |
| `assignment-2market/PDF report.pdf` | Written assignment report |
| `assignment-2market/PDF report.pptx` | Presentation slides |
| `assignment-2market/marketing_data.csv` | Raw marketing dataset |
| `assignment-2market/marketing_data_cleaned.csv` | Cleaned dataset used in analysis |
| `assignment-2market/metadata_2Market.txt` | Dataset documentation |

### Tools

- **Tableau Desktop** -- interactive dashboards and visual analytics
- **SQL / PostgreSQL / pgAdmin** -- database querying, joins, aggregation
- **Microsoft Excel** -- initial data exploration
