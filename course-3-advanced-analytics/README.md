# Course 3: Advanced Data Analytics (DA301)

## Assignment: Turtle Games Predictive Analytics

Built predictive models and performed NLP analysis for Turtle Games, a fictional games retailer. The project combined Python and R to deliver regression, classification, clustering, and sentiment analysis insights.

### Key Findings

- Developed regression models to predict customer spending behaviour
- Applied K-means clustering to segment customer groups
- Performed NLP sentiment analysis on customer reviews using NLTK
- Delivered complementary analyses in both Python and R
