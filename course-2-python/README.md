# Course 2: Data Analytics using Python (DA201)

## Assignment: NHS Appointments Diagnostic Analysis

Investigated NHS appointment utilisation data to identify trends, patterns, and capacity planning opportunities. Combined multiple datasets covering regional appointments, service settings, and national categories.

### Key Findings

- Analysed appointment trends across NHS service settings and regions
- Identified patterns in missed appointments and utilisation rates
- Delivered visualisations highlighting capacity gaps and seasonal trends

### Data Note

The large NHS datasets (`appointments_regional.csv`, `national_categories.xlsx`, `actual_duration.csv`) are excluded from this repository due to file size. They are publicly available from NHS England open data sources as documented in `metadata_nhs.txt`.
