# LSE Data Analytics Career Accelerator Portfolio

**Emmanuel Wagaba | Cohort 3 (2024-2025)**

This repository showcases my work from the LSE Data Analytics Career Accelerator programme -- a four-course programme covering business analytics, Python programming, machine learning, and a real-world employer capstone project.

---

## Key Projects

### 1. 2Market Marketing Analysis (DA101)

Analysed marketing campaign data for 2Market using SQL and Tableau to identify spending patterns by country, marital status, and product category. Delivered an interactive dashboard and a comprehensive report with actionable recommendations.

**Tools:** Tableau, SQL, PostgreSQL
**[View project](./course-1-business-analytics/assignment-2market/)**

### 2. NHS Appointments Diagnostic Analysis (DA201)

Investigated NHS appointment data to understand utilisation patterns across service settings, identify trends in missed appointments, and highlight capacity planning opportunities. Combined four datasets totalling 82+ MB for a multi-dimensional analysis.

**Tools:** Python, Pandas, Matplotlib, Seaborn
**[View project](./course-2-python/assignment-nhs-analysis/)**

### 3. Turtle Games Predictive Analytics (DA301)

Built predictive models for Turtle Games to forecast customer behaviour using regression, classification, and clustering techniques. Performed NLP sentiment analysis on customer reviews and delivered insights in both Python and R.

**Tools:** Python, R, scikit-learn, NLTK, ggplot2
**[View project](./course-3-advanced-analytics/assignment-turtle-games/)**

### 4. Study Group Student Data Analysis (DA401 -- Capstone)

Team 11 capstone project analysing Study Group Singapore's student data. Applied decision trees, logistic regression, random forest classification, K-means clustering, and time series forecasting to identify student outcome patterns and deliver actionable business recommendations.

**Tools:** Python, scikit-learn, Tableau, Pandas
**[View project](./course-4-employer-project/)**
