# Course 4: Employer Project (DA401) -- Capstone

## Project: Study Group Singapore Student Data Analysis

**Team 11** | LSE Data Analytics Career Accelerator

Capstone project analysing student data from Study Group's Singapore operations. The team delivered exploratory analysis, predictive models, and an interactive Tableau dashboard to support business decisions around student outcomes and engagement.

### Approach

Applied a multi-model analytical approach:

- **Exploratory Data Analysis** -- data profiling, feature distributions, missing value treatment
- **Decision Tree Classification** -- interpretable model for student outcome prediction
- **Logistic Regression** -- binary classification with feature importance analysis
- **Random Forest Classification** -- ensemble model with ROC curve evaluation
- **K-means Clustering** -- unsupervised segmentation of student groups
- **Time Series Analysis** -- temporal patterns in student data
